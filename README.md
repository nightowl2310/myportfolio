# Kavyansh Saxena's Portfolio Design

A modern UI/UX portfolio built using React, Vite, TypeScript, TailwindCSS, Framer Motion, and React Router.

This portfolio showcases selected projects, design case studies, smooth interactions, and responsive layouts with a clean dark-themed visual experience.

---

# ✨ Features

- Modern responsive UI
- Smooth animations using Framer Motion
- Dark / Light mode support
- Dedicated project showcase pages
- Interactive portfolio cards
- React Router based navigation
- Optimized Vite build setup
- Fully responsive layouts
- TailwindCSS styling system
